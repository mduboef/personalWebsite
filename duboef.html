Automated Boxing Judging System

Python TensorFlow Computer Vision Neural Networks

Developed the first-ever automated judging system for professional boxing using computer vision and neural networks. Scraped HD footage and scorecards from thousands of matches, using gradient descent to map statistical outputs onto judges' scores. The system is currently under review for official use by professional boxing commissions and represents a breakthrough in sports analytics.

NYC Subway Challenge

C++ Python Algorithms Graph Theory

Founded a cross-university open source project to find the fastest route through all 472 NYC subway stations with the goal of setting a world record. Built a C++ backend using BFS & A* algorithms on an adjacency matrix representation of the subway system, with Python scripts for data scraping from MTA and Google Maps APIs.

Have I Been Gerrymandered?

JavaScript Data Analysis Algorithms Political Science

Created an interactive online map that quantifies how gerrymandered individual congressional districts are. Developed a novel algorithm using electoral data that successfully identified Alabama districts later ruled unconstitutional by courts, validating the algorithm's effectiveness. The project combines political science with computer science to make gerrymandering analysis accessible.

Automatic Door Control System

Arduino Kotlin Bluetooth Android

Built a custom circuit with Arduino that connects a mobile app to existing door opener systems via Bluetooth, enabling disabled students at RPI to open campus doors remotely. Developed the Android application in Kotlin with an intuitive interface. This accessibility-focused project demonstrates how simple technology can have a meaningful impact on people's daily lives.
